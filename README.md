# cpp-risk

C++ implementations of fixed income and risk management tools.

## Projects

| Project | Description |
|---------|-------------|
| [fixed-income-risk-engine](fixed-income-risk-engine/) | Loads a bond portfolio from CSV, computes per-bond price, modified duration, convexity, DV01, and CS01, then runs scenario analysis across user-defined rate and spread shocks. Outputs base metrics, scenario P&L, and portfolio summary as CSV. |
